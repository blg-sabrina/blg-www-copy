制造模块

覆盖生产计划、控制与优化的完整制造执行系统

制造模块提供全面的制造执行系统（MES），覆盖从计划到执行的完整生产生命周期。本模块专为需要精细化生产控制、质量管理与制造优化能力的离散制造与流程制造企业而设计。

概述

制造模块提供：

  • 生产计划与排程 - 全面的产能与资源规划
  • 制造执行 - 实时生产控制与监控
  • 物料清单管理 - 复杂 BOM 结构与版本管理
  • 质量管理 - 过程质量控制与合规
  • 车间控制 - 工单管理与工时追踪
  • 制造分析 - 生产绩效与优化洞察
ℹ️
模块定位：本模块为离散制造企业、流程工业与混合模式生产环境提供完整的制造执行能力。

核心功能

生产计划与控制

  • 主生产计划 - 需求驱动的生产规划
  • 物料需求计划（MRP） - 自动化物料规划
  • 产能需求计划（CRP） - 资源与约束规划
  • 生产排程 - 有限与无限产能排程
  • 车间排程 - 实时工作中心排程

制造执行

  • 工单管理 - 完整的生产订单生命周期
  • 车间控制 - 实时生产监控与控制
  • 工时追踪 - 生产订单上的员工工时与出勤
  • 物料消耗 - 实时物料领用与倒冲
  • 生产报工 - 实时生产状态与完工汇报

物料清单（BOM）

  • 多层级 BOM - 复杂的层级化产品结构
  • BOM 版本管理 - 工程变更管理与版本控制
  • 虚拟 BOM - 直通装配件与计划 BOM
  • 替代 BOM - 多种生产方法与工艺路线
  • BOM 成本核算 - 标准与实际成本卷积计算

质量管理

  • 质量控制计划 - 按工序制定的质量规程
  • 过程检验 - 实时质量检查点
  • 统计过程控制（SPC） - 统计化质量监控
  • 不合格品管理 - 质量问题追踪与处置
  • 分析证书 - 质量文档与合规

核心应用

生产基础

内部工单管理
工单创建、排程与执行控制
流程监控应用
实时生产过程监控与控制

制造支持

内部库存调整应用
生产相关库存调整与废料管理
库存余额应用
实时在制品与产成品追踪

共享核心依赖

本模块依赖以下核心模块及其他模块的应用：

基础模块

  • 库存模块 - 原材料与产成品管理
  • 库存商品维护应用 - 产品与部件主数据
  • 组织应用 - 工厂与工作中心结构

计划与成本

  • 会计科目表应用 - 制造成本会计
  • 员工维护应用 - 人工资源管理
  • 供应商维护应用 - 原材料供应商管理

实施路径

第一阶段：基础搭建（第 1-3 周）

  1. 制造结构

    • 定义工厂与工作中心层级
    • 配置生产资源与产能
    • 设置工艺路线与工序定义
    • 定义质量控制点

  2. 产品工程

    • 创建物料清单结构
    • 定义工艺路线与作业指导书
    • 设置标准成本参数
    • 配置质量规格

第二阶段：计划与排程（第 4-6 周）

  1. 生产计划

    • 配置 MRP 参数与计划
    • 建立主生产计划
    • 落地产能需求计划
    • 配置排程算法

  2. 车间准备

    • 设置工单模板
    • 配置工时追踪系统
    • 落地领料发料流程
    • 建立生产报工机制

第三阶段：执行与优化（第 7-8 周）

  1. 制造执行

    • 落地车间控制系统
    • 配置实时数据采集
    • 建立质量控制规程
    • 落地绩效监控

  2. 上线与优化

    • 试生产运行与验证
    • 性能调优与优化
    • 用户培训与认证
    • 落地持续改进

业务流程

生产计划流程 

销售预测 → 主生产计划 → MRP 运算 → 产能规划 → 生产排程

制造执行流程 

工单下达 → 物料发放 → 生产加工 → 质量控制 → 工单完工

质量控制流程 

质控计划 → 过程检验 → 不合格品处理 → 纠正措施 → 质量证书
