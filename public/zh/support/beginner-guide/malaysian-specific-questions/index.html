马来西亚特色问题：初学者常见问答

了解 BigLedger 的马来西亚合规功能与本地业务要求

电子发票与 MyInvois

问：什么是 MyInvois，我什么时候需要使用它？

答： MyInvois 是马来西亚全新的强制性电子发票系统，最终将影响所有马来西亚企业：

MyInvois 对您的业务实际意味着什么：

  • 不再有纸质发票：所有发票必须是电子的
  • 直接提交给 LHDN：税务部门会立即收到副本
  • 标准化格式：所有发票必须遵循政府格式
  • 实时验证：系统在接受发票之前进行检查

实施时间表（您何时需要合规）：

第一阶段：2024 年 8 月（已经开始）

  • 年营业额超过 1 亿令吉的企业

第二阶段：2025 年 1 月（即将到来！）

  • 年营业额超过 2,500 万令吉的企业

第三阶段：2025 年 7 月（大多数小型企业）

  • 所有其余企业（包括非常小的企业）

这对小企业主意味着什么：

  • 如果您每年的收入低于 2,500 万令吉，您有时间到 2025 年 7 月
  • 如果您每年的收入低于 150,000 令吉，您仍然需要合规，但可能会获得简化流程
  • 重要：即使您规模很小，也要现在就开始准备！

为什么 BigLedger 非常适合这一点：

  • 我们已经获得 MDEC 的全面认证（政府批准）
  • 我们为您处理所有技术复杂性
  • 您只需正常创建发票——我们处理 MyInvois 的提交
  • 无需额外学习

小企业常见的疑虑：

“我只有 3 个客户——我真的需要这个吗？” → 是的，如果您是注册企业，您将需要合规

“我不擅长技术——这听起来太复杂了” → BigLedger 让它变得简单——您甚至不会察觉它在发生

“如果我不合规会怎样？” → 罚款从 200 令吉起，最高可达 20,000 令吉

“我现在用手写发票” → 您需要改用电子发票——BigLedger 让这个过渡变得简单

问：什么是 PEPPOL，我需要它吗？

答： PEPPOL 听起来很技术化，但可以把它想象成电子商业文件的"邮政系统"：

PEPPOL 实际上是什么：

  • 国际网络：就像电子邮件，但用于商业文件
  • 安全传递：保证发票和订单的传递
  • 全球标准：全球 40 多个国家使用
  • 业务效益：付款更快、错误更少、关系更好

您需要 PEPPOL 吗？

  • 需要，如果您销售给：政府机构、大型企业、国际客户
  • 可能需要，如果您销售给：中型马来西亚企业（他们可能会要求）
  • 可能还不需要，如果您销售给：个人消费者、非常小的企业

为什么 BigLedger 的 PEPPOL 认证很重要：

  • 面向未来：当您的客户需要时随时就绪
  • 无额外费用：PEPPOL 功能已包含在 BigLedger 中
  • 自动设置：我们处理所有技术配置
  • 竞争优势：您的许多竞争对手还没有这个

真实案例：

Ahmad 的空调服务：

  • 主要是住宅客户：现在不需要 PEPPOL
  • 获得政府大楼合同：需要 PEPPOL
  • 使用 BigLedger：只需勾选一个框，其他一切自动完成

Siti 的餐饮业务：

  • 供应企业活动：一些客户开始要求 PEPPOL
  • 使用 BigLedger：可以将电子发票作为高级服务提供

底线：您今天可能不需要 PEPPOL，但拥有它随时就绪能为您带来更多业务机会。

问：SST 在 BigLedger 中如何运作？

答： BigLedger 中的 SST（销售与服务税）旨在与马来西亚企业的实际运作方式完全匹配：

了解马来西亚 SST：

  • 销售税：大多数制成品征收 10%（由制造商/进口商支付）
  • 服务税：应税服务征收 6%（由服务提供商支付）
  • 谁支付：仅限超过一定门槛的企业
  • 申报：大多数企业每 2 个月一次

SST 门槛（您何时需要注册）：

  • 销售税：年营业额 > 500,000 令吉
  • 服务税：年营业额 > 150,000 令吉

BigLedger 如何自动处理 SST：

对于销售税：

  • 自动对您销售的制成品应用 10%
  • 追踪您欠马来西亚皇家关税局的金额
  • 自动生成 SST 申报表
  • 处理进口/出口场景

对于服务税：

  • 对符合条件的服务应用 6%
  • 追踪免税与应税服务
  • 计算每月义务
  • 管理服务税分组

在 BigLedger 中设置 SST：

  1. 在公司设置中注册您的 SST 号码
  2. 配置产品/服务税率（BigLedger 具有马来西亚默认值）
  3. 为不同类型的项目设置税组
  4. 启用自动税务计算

小企业常见的 SST 场景：

场景 1：小型餐厅（每年低于 150,000 令吉）

  • 无需 SST 注册
  • 但仍需追踪以进行门槛监控
  • BigLedger 在接近门槛时提醒您

场景 2：成长中的服务业务（每年超过 150,000 令吉）

  • 必须注册服务税
  • BigLedger 自动对应税服务应用 6%
  • 每 2 个月生成申报表

场景 3：小型制造商（每年超过 500,000 令吉）

  • 必须注册销售税
  • BigLedger 追踪制造与进口货物
  • 处理复杂的制造税务场景

初学者提示：

  • 尽早开始追踪：甚至在您需要注册之前
  • 使用 BigLedger 的默认值：已为马来西亚税率预先配置
  • 定期检查：每月检查 SST 报告
  • 专业建议：复杂情况请咨询会计师

马来西亚银行与付款

问：BigLedger 如何与马来西亚银行集成？

答： BigLedger 与所有主要马来西亚银行合作，让您的银行业务无缝衔接：

支持的马来西亚银行：

  • Maybank：全面集成，包括伊斯兰银行业务
  • CIMB：企业和商业账户
  • Public Bank：所有商业账户类型
  • Hong Leong Bank：标准和高级账户
  • RHB Bank：商业和商务账户
  • AmBank：包括面向中小企业的账户
  • Bank Islam：符合伊斯兰教法的银行业务
  • OCBC：本地和区域账户

银行集成意味着什么：

  • 自动交易导入：您的银行交易自动出现在 BigLedger 中
  • 实时余额：始终了解您的确切现金状况
  • 付款处理：直接从 BigLedger 发送付款
  • 对账：自动匹配交易
  • 多账户支持：轻松处理多个银行账户

设置银行集成：

方法 1：直接 API 连接（推荐）

  1. 授权 BigLedger 访问您的银行账户（只读）
  2. 银行交易每日自动下载
  3. 审查并分类导入的交易
  4. 自动对账在后台进行

方法 2：文件导入（如果直接连接不可用）

  1. 从您的网上银行下载银行对账单
  2. 上传到 BigLedger（支持所有马来西亚银行格式）
  3. 自动处理交易数据
  4. 手动分类新交易

马来西亚特有功能：

  • DuitNow 集成：处理二维码和即时付款
  • RENTAS 支持：实时全额结算追踪
  • IBG 处理：银行间 GIRO 交易处理
  • 支票管理：支持马来西亚支票格式

对马来西亚企业的好处：

  • 更快对账：以前需要数小时的工作现在只需几分钟
  • 更好的现金流可见性：实时了解现金状况
  • 减少错误：自动数据录入消除手动错误
  • 合规就绪：为审计和税务申报提供正确的记录

  • 自动汇率：每天从可靠来源更新

  • 历史汇率：追踪汇率随时间的变化

真实使用案例：

示例 1：进口业务

  4. 报告灵活性：以任何货币查看报告

汇率管理：

马来西亚税务影响：

  • 开支扣除：以适当汇率转换

本地合规与法规

问：我需要为 LHDN（内陆税收局）准备哪些报告？

答： BigLedger 自动生成 LHDN 要求的所有报告，让税务合规变得简单：

每月/每两月要求：

1. SST 申报（如已注册）：

  • 频率：每 2 个月
  • BigLedger 提供：包含所有计算的完整 SST-02 表格
  • 自动功能：税务计算、避免罚款、电子申报就绪

2. 每月税务扣除（MTD）（如适用）：

  • 频率：每月 15 日前
  • BigLedger 提供：用于员工税务扣除的 CP30 表格
  • 集成：与薪资模块连接

年度要求：

3. 所得税申报：

  • BE 表格：适用于企业（BigLedger 生成所有需要的附表）

4. 经审计的财务报表（如需要）：

  • 试算平衡表：自动生成

  • 审计追踪：为审计师提供完整文件

BigLedger 中可用的 LHDN 特定报告：

税务报告：

  • SST 计算工作表

  • 总分类账详情
  • 银行对账报告
  • 固定资产登记册
  • 库存估值报告

是什么让 BigLedger 符合 LHDN 要求：

  • 马来西亚会计准则：内置符合 FRS/MFRS
  • 税务计算：准确的马来西亚税务计算
  • 审计追踪：符合法律要求的完整记录保存
  • 保留：符合要求的 7 年文件存储
  • 语言支持：报告可用马来文和英文提供

转让定价（针对大型企业）：

税务合规最佳实践：

BigLedger 预防的常见合规错误：
