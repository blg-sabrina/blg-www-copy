📚 Bahagian 1: Log Masuk Pertama & Navigasi

Langkah 1: Akses Persekitaran Demo

1. Buka Laman Demo

  • Navigasi ke: https://demo-v1.bigledger.com
  • Anda akan melihat skrin log masuk BigLedger

2. Masukkan Kelayakan

  • Nama pengguna: demo-admin
  • Kata laluan: Demo2025!
  • Klik “Sign In”

3. Kejayaan Log Masuk Pertama

  • Papan pemuka dimuatkan dengan data sampel
  • Perhatikan menu utama di sebelah kiri
  • Bar atas menunjukkan pemberitahuan dan profil pengguna

Struktur Menu Utama

  • Dashboard: Skrin utama dengan KPI
  • Sales: Pengurusan pelanggan dan pesanan
  • Purchasing: Vendor dan perolehan
  • Inventory: Pengurusan stok
  • Accounting: Operasi kewangan
  • Reports: Analitik dan pandangan
  • Settings: Konfigurasi sistem

Petua Navigasi

  • Klik item menu untuk mengembangkan sub-menu
  • Gunakan breadcrumb untuk menjejaki lokasi
  • Tandakan bintang pada halaman yang kerap digunakan

Elemen Antara Muka Utama

  1. Bar Navigasi Atas

    • Pemilih syarikat (jika berbilang syarikat)
    • Carian pantas (Ctrl+K)
    • Loceng pemberitahuan
    • Menu profil pengguna

  2. Bar Sisi Kiri

    • Menu utama boleh runtuh
    • Navigasi modul
    • Kegemaran akses pantas

  3. Kawasan Kandungan Utama

    • Tajuk halaman dan tindakan
    • Grid data dan borang
    • Navigasi tab untuk data berkaitan

  4. Butang Tindakan

    • Tindakan utama (biru)
    • Tindakan sekunder (kelabu)
    • Tindakan bahaya (merah)

🔧 Bahagian 3: Konfigurasi Asas

Langkah 2: Tetapkan Keutamaan Anda

Navigasi ke: Settings > My Profile

  1. Maklumat Peribadi

    • Kemas kini nama paparan
    • Tetapkan gambar profil
    • Konfigurasikan maklumat hubungan

  2. Tetapan Serantau

    • Bahasa: Bahasa Inggeris
    • Format tarikh: MM/DD/YYYY
    • Paparan mata wang: USD
    • Format nombor: 1,234.56

  3. Simpan Perubahan

    • Klik “Save Profile”
    • Perubahan berkuat kuasa serta-merta

Navigasi ke: Settings > Display

  1. Pemilihan Tema

    • Mod terang (lalai)
    • Mod gelap
    • Auto (mengikut sistem)

  2. Pilihan Susun Atur

    • Paparan padat
    • Paparan selesa
    • Tetapan ketumpatan grid

  3. Keutamaan Senarai

    • Rekod setiap halaman: 25
    • Susunan isihan lalai
    • Keterlihatan lajur

Navigasi ke: Settings > Notifications

  1. Amaran E-mel

    • Pengesahan pesanan
    • Amaran stok rendah
    • Peringatan pembayaran

  2. Pemberitahuan Dalam Aplikasi

    • Tugasan tugas
    • Permintaan kelulusan
    • Kemas kini sistem

  3. Tetapan Kekerapan

    • Masa nyata
    • Ringkasan harian
    • Ringkasan mingguan

🚀 Bahagian 5: Transaksi Pertama Anda

Mencipta Pesanan Jualan Sampel

Menuju ke Pesanan Jualan

  1. Klik Sales dalam menu utama
  2. Pilih Sales Orders
  3. Anda akan melihat senarai pesanan sedia ada
  4. Klik butang New Sales Order (kanan atas)

Masukkan Butiran Pesanan

  1. Bahagian Pelanggan

    • Cari: Taip “ACME”
    • Pilih: “ACME Corporation”
    • Perhatikan alamat diisi secara automatik

  2. Tambah Item Baris

    • Klik “Add Item”
    • Cari: “PROD-001”
    • Kuantiti: 10
    • Harga diisi secara automatik

  3. Maklumat Pesanan

    • Tarikh penghantaran: Esok
    • Terma pembayaran: Net 30
    • Nota: “Demo order”

Simpan dan Proses

  1. Klik Save Draft

    • Nombor pesanan dijana
    • Status menunjukkan “Draft”

  2. Klik Submit for Approval

    • Status bertukar kepada “Pending”
    • Pemberitahuan dihantar

  3. Klik Approve (sebagai admin)

    • Status: “Approved”
    • Sedia untuk pemenuhan

Sahkan Kerja Anda

  1. Semak Senarai Pesanan

    • Pesanan anda muncul di bahagian atas
    • Status ditunjukkan dengan betul

  2. Lihat Butiran Pesanan

    • Klik nombor pesanan
    • Semak semua maklumat

  3. Semak Log Aktiviti

    • Lihat cap masa penciptaan
    • Semak sejarah kelulusan

📊 Bahagian 6: Pelaporan Asas

Menjana Laporan Pertama Anda

  1. Navigasi ke Reports

    • Klik “Reports” dalam menu utama
    • Pilih “Sales Reports”

  2. Pilih Jenis Laporan

    • Pilih “Sales Summary”
    • Tetapkan julat tarikh: Bulan Ini

  3. Konfigurasikan Pilihan

    • Kumpulkan mengikut: Pelanggan
    • Sertakan: Semua produk
    • Format: PDF

  4. Jana & Eksport

    • Klik “Generate Report”
    • Pratonton keputusan
    • Eksport mengikut keperluan

💡 Petua untuk Kejayaan Demo

Amalan Terbaik

Petua Kejayaan Demo:

  1. Mulakan mudah - Kuasai asas sebelum ciri lanjutan
  2. Gunakan data sampel - Jangan risau tentang membuat kesilapan

Soalan Lazim

S: Bolehkah saya merosakkan apa-apa dalam demo? J: Tidak! Demo ditetapkan semula setiap hari, jadi bereksperimen dengan bebas.

S: Bagaimana saya membuat asal perubahan? J: Kebanyakan tindakan mempunyai pilihan “undo” atau “reverse”. Semak menu tindakan.

S: Di manakah saya boleh mencari bantuan? J: Klik ikon (?) pada mana-mana halaman atau tekan F1 untuk bantuan kontekstual.
