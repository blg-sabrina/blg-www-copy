Complete applet catalog

This catalog lists every applet available in the BigLedger ecosystem, organized by module and functionality. Each applet is described with its primary purpose and key capabilities.

Understanding applets: Applets are reusable functional components that can belong to multiple modules. This many-to-many relationship eliminates duplication and ensures consistent behavior across the entire BigLedger system.

Implementation roadmap

  1. Phase 1 — Foundation. Implement all 13 Core module applets as the foundation for all operations.
  2. Phase 2 — Primary operations. Add specialized module applets based on business requirements: Accounting for financial operations, Sales & CRM for sales, and warehouse applets for inventory.
  3. Phase 3 — Advanced features. Implement advanced applets: Commerce for e-commerce integration, ERP for multi-entity operations, and E-Invoice for compliance requirements.
  4. Phase 4 — Optimization. Add analytics, automation, and optimization applets: Digital CRM for business intelligence, Workflow for process automation, and analytics for performance monitoring.
