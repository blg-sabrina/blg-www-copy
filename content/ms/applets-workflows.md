Seni bina applet modular BigLedger membolehkan perniagaan membina penyelesaian tersuai dengan menggabungkan applet khusus yang berfungsi secara lancar bersama-sama melalui hab data pusat kami.

## Memahami Applet

### Apakah itu Applet?

Applet ialah modul perniagaan mandiri yang:
- Fokus pada fungsi perniagaan tertentu
- Berkongsi data melalui hab pusat
- Boleh beroperasi secara bebas atau bersepadu
- Berskala berdasarkan keperluan perniagaan
- Dikemas kini tanpa menjejaskan applet lain

## Senarai Applet Lengkap Mengikut Kategori

Berdasarkan seni bina applet komprehensif BigLedger, platform kami menawarkan applet khusus yang disusun ke dalam lapan kategori teras:

### 🏢 Applet Data Induk

Applet asas penting untuk konfigurasi sistem:

- **Organization** - Pengurusan struktur dan hierarki syarikat
- **Chart Of Account** - Persediaan struktur akaun kewangan
- **Cashbook** - Pengurusan dan penjejakan tunai
- **Employee** - Pangkalan data dan pengurusan pekerja
- **Doc Item** - Data induk dokumen dan item
- **Customer** - Pengurusan data induk pelanggan
- **Supplier** - Pangkalan data vendor/pembekal
- **Pricebook** - Peraturan harga dan katalog
- **Entity Maintenance** - Pengurusan hubungan entiti
- **Permission Wizard** - Kawalan akses dan kebenaran pengguna
- **Voucher** - Konfigurasi jenis baucar

### 🛒 Applet Aliran Kerja Pembelian

## Orkestrasi Aliran Kerja

### Aliran Kerja Silang Applet

#### Order-to-Cash (O2C)
```
CRM → Quote → Sales Order → Inventory Check → Delivery → E-Invoice → Payment → Bank Reconciliation → GL Posting
```

**Applet Terlibat:**
- CRM
- Sales Order Management
- Inventory
- Warehouse
- E-Invoice
- Accounts Receivable
- Bank Reconciliation
- General Ledger

#### Procure-to-Pay (P2P)
```
Requisition → RFQ → PO → Goods Receipt → Invoice Matching → Payment → Bank Reconciliation → GL Posting
```

**Applet Terlibat:**
- Procurement
- Vendor Management
- Inventory
- Accounts Payable
- E-Invoice
- Bank Integration
- General Ledger

#### Record-to-Report (R2R)
```
Transaction Entry → Validation → Posting → Reconciliation → Adjustments → Closing → Reporting → Analysis
```

**Applet Terlibat:**
- General Ledger
- Sub-ledgers
- Bank Reconciliation
- Fixed Assets
- Financial Reporting
- Business Intelligence
