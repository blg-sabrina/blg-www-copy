## Gambaran Keseluruhan Antara Muka

Antara muka moden BigLedger direka untuk kecekapan dan kemudahan penggunaan, menyediakan akses pantas ke semua fungsi perniagaan melalui sistem navigasi yang intuitif.

## Menu Navigasi Utama

### Bar Navigasi Atas
- **Modul**: Akses modul perniagaan teras (Perakaunan, Inventori, CRM, dll.)
- **Applet**: Fungsi lanjutan dan aliran kerja tersuai
- **E-Invois**: Integrasi PEPPOL dan MyInvois
- **Operasi Perniagaan**: Tugas operasi harian
- **Panduan Pengguna**: Dokumentasi menyeluruh

### Bar Alat Akses Pantas
- **Cari**: Carian global merentasi semua modul
- **Pemberitahuan**: Amaran dan kemas kini masa nyata
- **Profil Pengguna**: Tetapan akaun dan keutamaan
- **Bantuan**: Bantuan sensitif konteks

## Susun Atur Ruang Kerja

### Widget Papan Pemuka
- Penempatan widget boleh disesuaikan
- Fungsi seret-dan-lepas
- Kemas kini data masa nyata
- Pelbagai paparan papan pemuka

### Grid Data
- Penyesuaian lajur
- Penapisan lanjutan
- Keupayaan eksport
- Penyuntingan sebaris

## Pintasan Papan Kekunci

### Navigasi
- `Ctrl + K`: Carian pantas
- `Ctrl + /`: Togol menu
- `Ctrl + M`: Penukar modul
- `Esc`: Tutup dialog

### Operasi Data
- `Ctrl + N`: Rekod baharu
- `Ctrl + S`: Simpan
- `Ctrl + D`: Padam
- `Ctrl + E`: Mod suntingan

## Navigasi Mudah Alih

### Reka Bentuk Responsif
- Antara muka dioptimumkan untuk sentuhan
- Sokongan gerak isyarat leretan
- Keupayaan mod luar talian
- Ciri aplikasi web progresif

## Petua untuk Navigasi Cekap

1. **Gunakan Kegemaran**: Sematkan halaman yang kerap digunakan
2. **Sesuaikan Papan Pemuka**: Susun widget untuk aliran kerja anda
3. **Pelajari Pintasan**: Percepatkan tugas biasa
4. **Gunakan Carian**: Cari dengan pantas sebarang fungsi atau data
5. **Tetapkan Keutamaan**: Konfigurasi antara muka mengikut keperluan anda

## Mendapatkan Bantuan

- **Bantuan Kontekstual**: Klik ikon ? pada mana-mana halaman
- **Panduan Pengguna**: Akses dokumentasi menyeluruh
- **Portal Sokongan**: Serahkan tiket dan jejaki isu
- **Forum Komuniti**: Berhubung dengan pengguna lain
