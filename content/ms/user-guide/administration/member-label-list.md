5.Senarai label ahli: Cipta Senarai Label Ahli>Maklumat wajib>Sunting
5.1. Sila klik 'Senarai label ahli' daripada senarai modul dan klik butang '+' untuk mencipta senarai label ahli baharu.
5.2. Sila isikan maklumat wajib (*) untuk mencipta senarai label ahli. Selepas mengisi semua maklumat Wajib (*), sila klik 'Create' untuk menambah Senarai label ahli baharu. Maklumat wajib:
Kod senarai label
Nama senarai label
5.3. Selepas menambah senarai label ahli, ia akan disenaraikan di halaman utama. Sila klik pada ahli untuk menyuntingnya. 
Anda boleh menyunting butiran dan label senarai label ahli.
5.3.1. Dalam tab butiran, maklumat berikut boleh disunting:
Nama senarai label
Status
Selepas menukar butiran medan, sila klik 'save'.
Sila klik 'delete' untuk mengeluarkan senarai label ahli daripada senarai.
5.3.2. Dalam tab label, anda boleh menambah label ke senarai label. 
Sila klik butang '+' untuk menambah label.
Sila isikan maklumat wajib untuk menambah label baharu. Sila ambil perhatian bahawa apabila anda mencipta label baharu dalam senarai label, label tersebut akan disenaraikan dalam modul label ahli.
