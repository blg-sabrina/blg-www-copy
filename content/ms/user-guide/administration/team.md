2. Pasukan: Cipta Pasukan > Maklumat Wajib> Sunting (Maklumat umum, Ahli, Import Pengguna, Kebenaran)
2.1 Sila klik butang '+' untuk menambah pasukan baharu.
2.2 Sila isikan maklumat wajib (*) untuk mencipta pasukan. Maklumat wajib:
Nama Pasukan
Kod Pasukan
E-mel Pasukan
Sila pilih Keterlihatan Pasukan daripada senarai.
Sila pilih Penyertaan Pasukan daripada senarai
Selepas mengisi semua maklumat Wajib (*), sila klik 'Create' untuk menambah pasukan baharu.
2.3 Selepas menambah pasukan, ia akan disenaraikan di halaman utama. Sila klik pada pasukan untuk menyuntingnya.
2.4 Sila sunting maklumat umum Pasukan daripada tab 'Edit general info'.
Hanya medan berikut boleh disunting:
Nama Pasukan
Penerangan Pasukan
Keterlihatan Pasukan
Penyertaan Pasukan
Status Pasukan
2.5 Sila tambah atau sunting ahli pasukan daripada tab 'Edit Members'.
2.5.1.Sila klik butang '+' untuk menambah pengguna kepada pasukan. 
Sila cari e-mel pengguna-Setelah pengguna mendaftar di http://akaun.com , e-mel mereka akan disenaraikan dalam 'Select Email'. 
Sila pilih pangkat-Sila 'Select Rank' untuk membezakan peranan ahli. 
Sila pilih status
Sila klik butang 'Add' untuk menambah pengguna kepada pasukan.
2.5.2. Selepas menambah ahli dalam pasukan, sila klik Ahli daripada senarai untuk menyunting atau Memadamkannya.
2.6 Sila import Pengguna kepada Pasukan daripada tab 'Import Users'. 
Sila ikut arahan atau muat turun 'simple format' untuk memuat naik fail yang betul.
Sila klik 'Upload file' untuk mengimport pengguna dalam pasukan
2.7 Sila tambah atau sunting kebenaran daripada tab 'Edit Permission'.
Anda boleh 'Select permission' daripada senarai untuk Menambah kebenaran.
Selepas menambah kebenaran, anda boleh Menyunting atau Memadamkannya daripada 'Permission.
