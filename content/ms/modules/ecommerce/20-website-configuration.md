# 2.0 Konfigurasi Laman Web﻿
**Page ID**: 3187343370
**Last Updated**: 2024-11-17T11:07:01.283Z
## 2.1 Menyediakan Laman Web Anda﻿
CP-Commerce menawarkan antara muka mesra pengguna untuk mengkonfigurasi laman web anda. Rujuk bahagian Konfigurasi Laman Web untuk arahan terperinci tentang menyediakan dan menyesuaikan laman web anda.
## 2.2 Menyesuaikan Kandungan Laman Web﻿
Ketahui cara menyesuaikan kandungan laman web anda, termasuk penyenaraian produk, imej, penerangan, dan penetapan harga, untuk mewujudkan pengalaman membeli-belah dalam talian yang menarik bagi pelanggan anda.
