Create - digunakan untuk menyegerakkan item yang dipilih ke dalam kedai pasaran (marketplace). Create terdiri daripada 2 jenis:
Create Single - digunakan apabila pengguna ingin menyegerakkan 1 (satu) item ke dalam pasaran 
Create single item terdiri daripada 2 tab:
Store Tab - digunakan untuk memilih kedai yang ingin disegerakkan oleh pengguna di mana pengguna boleh memilih berbilang kedai.
Setelah Store dipilih, pengguna perlu memilih kategori:
2. Item Tab - digunakan untuk memilih item semasa yang sedang anda sunting dan ingin disegerakkan ke pasaran:
Setelah kedai dan kategori dipilih, pengguna perlu menekan butang "start listing" yang akan membawa ke halaman ringkasan:
Pengguna mesti klik pada item dan mengisi semua butiran atribut:
Setelah pengguna selesai mengisi butiran atribut, mereka boleh klik pada butang "Submit" untuk menyegerakkan ke pasaran.
