مرجع API كامل لخدمات BigLedger.

## واجهات API المتاحة

### واجهات API الأساسية
- **Transaction API** - إنشاء المعاملات والاستعلام عنها وإدارتها
- **Account API** - إدارة الحسابات والمصادقة
- **Wallet API** - عمليات المحفظة والاستعلام عن الأرصدة

### واجهات API المتقدمة
- **Smart Contract API** - نشر العقود الذكية والتفاعل معها
- **Analytics API** - الوصول إلى تحليلات المعاملات والتقارير
- **Webhook API** - تكوين وإدارة خطافات الويب (Webhooks)
