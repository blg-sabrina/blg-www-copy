## 🔨 سير العمل 1: إدارة قوائم المواد {#bom-setup}

تعلّم إنشاء وإدارة قوائم المواد متعددة المستويات مع التحكم في الإصدارات وإدارة التغييرات الهندسية.

### السيناريو: إنشاء قائمة مواد لـ AC-SENSOR-001

**الهدف**: إعداد هيكل قائمة مواد كامل لمستشعر درجة حرارة السيارات بما في ذلك التجميعات الفرعية وعمليات التوجيه.

**النتيجة المتوقعة**: قائمة مواد متعددة المستويات مُتحقّق منها وجاهزة لتخطيط الإنتاج والتكلفة.

### تعليمات خطوة بخطوة

{{< tabs items="الإعداد,المكونات,التوجيه,التحقق" >}}
  {{< tab >}}
  **1. الوصول إلى إدارة قوائم المواد**

  1. سجّل الدخول إلى البيئة التجريبية
  2. انتقل إلى **التصنيع** ← **قائمة المواد**
  3. انقر على **+ قائمة مواد جديدة**
  4. اختر **المنتج**: AC-SENSOR-001
  5. عيّن **نوع قائمة المواد**: تصنيع
  6. أدخل **الإصدار**: V1.0
  7. عيّن **تاريخ السريان**: التاريخ الحالي

  **النتيجة المتوقعة**: تم إنشاء رأس قائمة مواد جديد مع تحميل تفاصيل المنتج.
  {{< /tab >}}

  {{< tab >}}
  **2. إضافة مكونات قائمة المواد**

  **مكونات التجميعة الرئيسية**:
  1. انقر على **إضافة مكوّن**
  2. أضف هذه الأصناف مع الكميات:
     - شريحة المستشعر (IC-TEMP-DS18): 1 وحدة
     - لوحة الدائرة المطبوعة (PCB-SENSOR-01): 1 وحدة
     - مقاومة 10K (R-10K-0805): 2 وحدة
     - مكثف 100nF (C-100N-0805): 1 وحدة
     - موصّل 3 دبابيس (CON-3PIN-JST): 1 وحدة
     - هيكل بلاستيكي (HSG-PLAST-01): 1 وحدة

  3. عيّن **وحدة القياس** لكل مكوّن
  4. حدّد **نسبة الهالك** (عادةً 2-5% للإلكترونيات)
  5. عيّن **وقت التوريد** لكل مكوّن

  **النتيجة المتوقعة**: قائمة مكونات كاملة مع الكميات والمواصفات.
  {{< /tab >}}

  {{< tab >}}
  **3. تحديد عمليات التوجيه**

  1. انقر على علامة تبويب **التوجيه**
  2. أضف هذه العمليات بالتسلسل:
     - **العملية 10**: تجميع SMT (مركز العمل: SMT-LINE-01)
     - **العملية 20**: الإدراج عبر الثقوب (مركز العمل: ASSEMBLY-01)
     - **العملية 30**: اللحام (مركز العمل: SOLDER-01)
     - **العملية 40**: الاختبار (مركز العمل: TEST-STATION-01)
     - **العملية 50**: تجميع الهيكل (مركز العمل: ASSEMBLY-02)
     - **العملية 60**: الفحص النهائي (مركز العمل: QC-STATION-01)

  3. عيّن **وقت الإعداد** و**وقت التشغيل** لكل عملية
  4. حدّد معدلات **تكلفة العمالة** و**تكلفة الآلة**

  **النتيجة المتوقعة**: توجيه كامل مع معايير الوقت ومعلومات التكلفة.
  {{< /tab >}}

  {{< tab >}}
  **4. التحقق من قائمة المواد**

  1. انقر على **التحقق من قائمة المواد**
  2. يتحقق النظام من:
     - توفر المكونات
     - مصادر الموردين
     - اكتمال التكلفة
     - اتساق التوجيه

  3. راجع نتائج التحقق
  4. أصلح أي تحذيرات أو أخطاء
  5. انقر على **الموافقة على قائمة المواد**
  6. عيّن الحالة إلى **نشط**

  **النتيجة المتوقعة**: قائمة مواد مُتحقّق منها ومعتمدة وجاهزة للاستخدام في الإنتاج.
  {{< /tab >}}
{{< /tabs >}}

### النصائح وأفضل الممارسات

{{< callout type="tip" >}}
**نصائح إدارة قوائم المواد**:
- استخدم أوصاف مكونات وصفية تتضمن المواصفات
- احتفظ بقوائم استبدال المكونات لتحقيق المرونة
- أعدّ التحكم في التغييرات الهندسية لمراجعات قوائم المواد
- ضمّن عوامل الهالك بناءً على البيانات التاريخية
- المراجعة والتحديث المنتظم لأوقات التوريد
{{< /callout >}}

### المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|-------|-------|----------|
| المكوّن غير موجود | سجل صنف رئيسي مفقود | أنشئ سجل الصنف الرئيسي أولًا في وحدة المخزون |
| مرجع دائري في قائمة المواد | المكوّن يشير إلى العنصر الأصل | تحقق من هيكل قائمة المواد بحثًا عن الحلقات |
| التكلفة غير مكتملة | بيانات التكلفة مفقودة | حدّث التكاليف المعيارية في سجل الصنف الرئيسي |
| فشل التحقق من التوجيه | مركز العمل غير محدّد | أعدّ مراكز العمل في إعداد التصنيع |

## 📊 سير العمل 2: تخطيط الإنتاج وتخطيط متطلبات المواد {#production-planning}

أتقن تخطيط متطلبات المواد (MRP) لتحسين جداول الإنتاج ومستويات المخزون.

### السيناريو: تخطيط الإنتاج للأوامر الشهرية

**الهدف**: تشغيل تخطيط متطلبات المواد لإنشاء توصيات الشراء وأوامر الإنتاج للطلب المتوقع للشهر القادم.

**النتيجة المتوقعة**: جدول إنتاج محسّن مع خطة توريد المواد.

### تعليمات خطوة بخطوة

{{< tabs items="إعداد الطلب,تشغيل MRP,التحليل,الإجراءات" >}}
  {{< tab >}}
  **1. إعداد مصادر الطلب**

  1. انتقل إلى **التخطيط** ← **إدارة الطلب**
  2. راجع **أوامر المبيعات** التي تتطلب إنتاجًا:
     - SO-2025-001: 100 وحدة AC-SENSOR-001 (الاستحقاق: الأسبوع 2)
     - SO-2025-002: 50 وحدة PC-BOARD-LCD (الاستحقاق: الأسبوع 3)
     - SO-2025-003: 200 وحدة SW-MODULE-RF (الاستحقاق: الأسبوع 4)

  3. تحقق من **توقعات الطلب**:
     - انقر على **توقعات الطلب**
     - راجع الكميات المتوقعة حسب المنتج
     - عدّل التوقعات عند الحاجة

  4. تحقق من **مستويات مخزون الأمان**:
     - انتقل إلى **المخزون** ← **الصنف الرئيسي**
     - تحقق من إعدادات مخزون الأمان للبضائع الجاهزة

  **النتيجة المتوقعة**: صورة كاملة للطلب تشمل الأوامر المؤكدة والتوقعات.
  {{< /tab >}}

  {{< tab >}}
  **2. تنفيذ تشغيل MRP**

  1. انتقل إلى **التخطيط** ← **تخطيط MRP**
  2. انقر على **تشغيل MRP جديد**
  3. عيّن **معلمات التخطيط**:
     - أفق التخطيط: 12 أسبوعًا
     - حجم الفترة: أسبوعي
     - تضمين مخزون الأمان: نعم
     - مراعاة تقويم العمل: نعم

  4. اختر **المنتجات المراد تخطيطها**:
     - جميع البضائع الجاهزة
     - التجميعات الفرعية الرئيسية
     - المواد الخام الحرجة

  5. انقر على **تشغيل MRP**
  6. راقب التقدم في **حالة تشغيل MRP**

  **النتيجة المتوقعة**: تشغيل MRP مكتمل مع إنشاء توصيات التوريد.
  {{< /tab >}}

  {{< tab >}}
  **3. تحليل نتائج MRP**

  1. افتح **لوحة معلومات نتائج MRP**
  2. راجع **رسائل الاستثناءات**:
     - الأوامر المتأخرة
     - قيود الطاقة
     - نقص المواد
     - انتهاكات السياسة

  3. تحقق من **الأوامر المخططة**:
     - أوامر الإنتاج المراد إنشاؤها
     - أوامر الشراء المطلوبة
     - التوقيت والكميات

  4. راجع **تحليل الطاقة**:
     - تحميل مركز العمل
     - تحديد نقاط الاختناق
     - متطلبات العمل الإضافي

  **النتيجة المتوقعة**: فهم واضح لمتطلبات الإنتاج والقيود.
  {{< /tab >}}

  {{< tab >}}
  **4. اتخاذ إجراءات التخطيط**

  **إنشاء أوامر الإنتاج**:
  1. اختر أوامر الإنتاج المخططة
  2. انقر على **إنشاء أوامر الإنتاج**
  3. راجع وأكّد تفاصيل الأمر
  4. عيّن مستويات الأولوية

  **إنشاء طلبات الشراء**:
  1. اختر متطلبات المواد
  2. انقر على **إنشاء طلبات الشراء**
  3. عيّنها إلى المشترين المناسبين
  4. عيّن تواريخ التسليم المطلوبة

  **التعامل مع الاستثناءات**:
  1. راجع أحمال الطاقة الزائدة
  2. أعد جدولة الأوامر غير الحرجة
  3. انظر في خيارات التعاقد من الباطن
  4. عدّل مخزون الأمان عند الحاجة

  **النتيجة المتوقعة**: خطة إنتاج وتوريد قابلة للتنفيذ وجاهزة للتنفيذ.
  {{< /tab >}}
{{< /tabs >}}

### ميزات MRP المتقدمة

{{< callout type="info" >}}
**خيارات تخطيط MRP**:
- **MRP التغيير الصافي**: إعادة حساب الأصناف المتغيرة فقط
- **MRP التجديدي**: إعادة حساب كاملة لجميع الأصناف
- **تخطيط الطاقة المحدودة**: مراعاة قيود مركز العمل
- **التخطيط متعدد المواقع**: التخطيط عبر مواقع متعددة

## 🏗️ سير العمل 3: إدارة أوامر العمل {#work-orders}

تعلّم إنشاء وتتبع وإكمال أوامر عمل الإنتاج مع تتبع المواد والعمالة.

### السيناريو: معالجة أمر إنتاج لـ AC-SENSOR-001

**الهدف**: تنفيذ دورة حياة أمر العمل الكاملة من الإنشاء إلى استلام البضائع.

**النتيجة المتوقعة**: إنتاج بضائع جاهزة ونقلها إلى المخزون مع تتبع دقيق للتكلفة.

### تعليمات خطوة بخطوة

{{< tabs items="الإنشاء,صرف المواد,الإنتاج,الإكمال" >}}
  {{< tab >}}
  **1. إنشاء أمر عمل**

  1. انتقل إلى **التصنيع** ← **أوامر الإنتاج**
  2. انقر على **+ أمر إنتاج جديد**
  3. أدخل تفاصيل الأمر:
     - **المنتج**: AC-SENSOR-001
     - **الكمية**: 100 وحدة
     - **تاريخ الاستحقاق**: الجمعة القادمة
     - **الأولوية**: عادية

  4. يملأ النظام تلقائيًا:
     - مكونات قائمة المواد
     - عمليات التوجيه
     - التكاليف المعيارية

  5. راجع وأكّد الأمر
  6. انقر على **إصدار الأمر**
  7. تتغير حالة الأمر إلى **صادر**

  **النتيجة المتوقعة**: تم إنشاء أمر العمل وهو جاهز لتخطيط المواد.
  {{< /tab >}}

  {{< tab >}}
  **2. عملية صرف المواد**

  **صرف المواد إلى الإنتاج**:
  1. انقر على **صرف المواد** من أمر العمل
  2. راجع متطلبات المواد:
     - الكمية المطلوبة مقابل المتاحة
     - متطلبات الدفعات/الأرقام التسلسلية
     - فحوصات تاريخ انتهاء الصلاحية

  3. لكل مكوّن:
     - اختر موقع المخزون
     - اختر دفعات محددة عند الطلب
     - أكّد الكميات

  4. انقر على **صرف جميع المواد**
  5. يحدّث النظام:
     - أرصدة المخزون
     - حالة أمر العمل
     - تكاليف المواد

  **النتيجة المتوقعة**: تم تخصيص جميع المواد لأمر الإنتاج.
  {{< /tab >}}

  {{< tab >}}
  **3. تتبع الإنتاج**

  **الإبلاغ عن العمليات**:
  1. انتقل إلى **أرضية الإنتاج** ← **الإبلاغ عن العمليات**
  2. اختر أمر العمل والعملية
  3. أبلغ عن تقدم الإنتاج:
     - **الكمية المكتملة**: أدخل الإنتاج الفعلي
     - **ساعات العمالة**: سجّل الوقت الفعلي المستغرق
     - **ساعات الآلة**: سجّل استخدام المعدات
     - **العيوب**: سجّل أي مشكلات جودة

  4. لكل عملية (العملية 10-60):
     - ابدأ العملية
     - سجّل الإكمال
     - انتقل إلى العملية التالية

  5. يتتبع النظام:
     - التقدم في الوقت الفعلي
     - التكاليف الفعلية مقابل المعيارية
     - حسابات الكفاءة

  **النتيجة المتوقعة**: تتبع إنتاج كامل مع تراكم التكلفة.
  {{< /tab >}}

  {{< tab >}}
  **4. إكمال الأمر**

  **الخطوات النهائية**:
  1. أكمل العملية النهائية (العملية 60)
  2. سجّل نتائج الفحص النهائي
  3. انقر على **إكمال الأمر**
  4. اختر نوع الإكمال:
     - **الإكمال الكامل**: تم إنتاج جميع الكميات
     - **الإكمال الجزئي**: بعض التباين في الكمية

  5. **استلام البضائع**:
     - حدّد موقع التخزين
     - أنشئ أرقام الدفعات/الأرقام التسلسلية
     - حدّث أرصدة المخزون
     - احسب تكلفة المنتج النهائية

  6. **إغلاق الأمر**:
     - راجع تباينات التكلفة
     - أغلق مصروفات المواد المتبقية
     - حدّث إحصائيات الإنتاج
     - أرشف مستندات الأمر

  **النتيجة المتوقعة**: بضائع جاهزة في المخزون بتكلفة معيارية دقيقة.
  {{< /tab >}}
{{< /tabs >}}

### لوحة معلومات مراقبة الإنتاج

## 🔍 سير العمل 4: إدارة مراقبة الجودة {#quality-control}

نفّذ إدارة جودة شاملة تشمل الفحص الوارد والاختبار أثناء العملية والفحص النهائي.

### السيناريو: مراقبة الجودة للمكونات الإلكترونية

**الهدف**: تنفيذ سير عمل جودة كامل من فحص المواد الواردة إلى اعتماد المنتج النهائي.

**النتيجة المتوقعة**: سجلات جودة محفوظة مع إمكانية تتبع كاملة ووثائق الامتثال.

### تعليمات خطوة بخطوة

{{< tabs items="مراقبة الجودة الواردة,أثناء العملية,الفحص النهائي,الشهادات" >}}
  {{< tab >}}
  **1. فحص المواد الواردة**

  **إعداد خطط الفحص**:
  1. انتقل إلى **الجودة** ← **خطط الفحص**
  2. أنشئ خطة لـ **IC-TEMP-DS18** (شريحة مستشعر درجة الحرارة)
  3. حدّد معايير الفحص:
     - **الفحص البصري**: سلامة التغليف
     - **الاختبار الكهربائي**: قياس المقاومة
     - **اختبار الأداء الوظيفي**: دقة درجة الحرارة
     - **التوثيق**: شهادة الامتثال

  **تنفيذ الفحص الوارد**:
  1. انتقل إلى **الجودة** ← **أوامر الفحص**
  2. اختر الشحنة الواردة
  3. أنشئ أمر فحص
  4. سجّل نتائج الاختبار:
     - حجم العينة: 10 قطع من الدفعة
     - معايير النجاح/الفشل لكل اختبار
     - إدخال بيانات القياس

  **النتيجة المتوقعة**: مواد معتمدة وجاهزة للاستخدام في الإنتاج.
  {{< /tab >}}

  {{< tab >}}
  **2. مراقبة الجودة أثناء العملية**

  **إعداد نقاط تفتيش العملية**:
  1. حدّد نقاط تفتيش الجودة في التوجيه:
     - **بعد SMT**: فحص وصلات اللحام
     - **بعد التجميع**: الفحوصات الأبعادية
     - **بعد الاختبار**: التحقق الوظيفي

  **تنفيذ الفحص أثناء العملية**:
  1. عند كل نقطة تفتيش:
     - سجّل المشغّل الذي يجري الفحص
     - وثّق نتائج القياس
     - دوّن أي عيوب مكتشفة
     - طبّق الإجراءات التصحيحية

  2. **التحكم الإحصائي في العملية**:
     - راقب مخططات التحكم
     - تتبّع قدرة العملية
     - حدّد الاتجاهات والأنماط
     - أطلق تنبيهات لظروف الخروج عن السيطرة

  **النتيجة المتوقعة**: مراقبة جودة العملية والتحكم فيها طوال الإنتاج.
  {{< /tab >}}

  {{< tab >}}
  **3. فحص المنتج النهائي**

  **اختبار نهائي شامل**:
  1. انتقل إلى **الجودة** ← **الفحص النهائي**
  2. اختر أمر العمل المكتمل
  3. نفّذ تسلسل الاختبار:
     - **اختبار الأداء الوظيفي**: فحص تشغيلي كامل
     - **المعايرة**: التحقق من دقة المستشعر
     - **الاختبار البيئي**: دورة درجة الحرارة
     - **اختبار المتانة**: مقاومة الاهتزاز

  **جمع بيانات الجودة**:
  1. سجّل القياسات التفصيلية
  2. سجّل معدات الاختبار المستخدمة
  3. وثّق الظروف البيئية
  4. دوّن اعتماد المفتش
  5. أنشئ مقاييس الجودة

  **قرار التصرف**:
  - **قبول**: نقل إلى البضائع الجاهزة
  - **رفض**: إعادة إلى إعادة التصنيع
  - **تعليق**: بانتظار تقييم إضافي

  **النتيجة المتوقعة**: منتجات مضمونة الجودة وجاهزة للشحن.
  {{< /tab >}}

  {{< tab >}}
  **4. شهادات الجودة وإمكانية التتبع**

  **إنشاء الشهادات**:
  1. انتقل إلى **الجودة** ← **الشهادات**
  2. اختر المنتجات للاعتماد
  3. أنشئ **شهادة الامتثال**:
     - استيفاء مواصفات المنتج
     - ملخص نتائج الاختبار
     - امتثال نظام الجودة
     - التوقيعات المعتمدة

  **توثيق إمكانية التتبع**:
  1. **إمكانية تتبع المواد**:
     - أرقام دفعات المكونات
     - معلومات المورد
     - تواريخ الاستلام

  2. **إمكانية تتبع العملية**:
     - تفاصيل أمر الإنتاج
     - سجلات المشغّل
     - المعدات المستخدمة
     - معلمات العملية

  **تقارير الامتثال**:
  - مقاييس الجودة الشهرية
  - تقارير الجودة للعملاء
  - وثائق الامتثال التنظيمي
  - تتبع الإجراءات التصحيحية

  **النتيجة المتوقعة**: حزمة وثائق جودة كاملة للتسليم إلى العميل.
  {{< /tab >}}
{{< /tabs >}}

### أفضل ممارسات مراقبة الجودة

{{< callout type="warning" >}}

## 💰 سير العمل 5: تقييم المخزون وتحليل التكلفة {#inventory-valuation}

أتقن طرق تكلفة المخزون وتحليل تكلفة الإنتاج للحصول على تقارير مالية دقيقة.

### السيناريو: تحليل التكلفة والتقييم في نهاية الشهر

**الهدف**: تنفيذ إجراءات التكلفة في نهاية الشهر بما في ذلك تقييم المخزون وتحليل التباين وتقارير التكلفة.

**النتيجة المتوقعة**: تكاليف منتجات دقيقة وتقييم مخزون للبيانات المالية.

### تعليمات خطوة بخطوة

{{< tabs items="طرق التكلفة,تحليل التباين,تقييم المخزون,تقارير التكلفة" >}}
  {{< tab >}}
  **1. تكوين طرق التكلفة**

  **إعداد التكلفة المعيارية**:
  1. انتقل إلى **المحاسبة** ← **تكلفة المنتج**
  2. اختر المنتج **AC-SENSOR-001**
  3. أعدّ **عناصر التكلفة المعيارية**:
     - **تكلفة المواد**: 12.50 دولار (من قائمة المواد)
     - **تكلفة العمالة**: 3.20 دولار (من التوجيه)
     - **تكلفة النفقات العامة**: 2.80 دولار (مخصصة)
     - **إجمالي المعياري**: 18.50 دولار

  **تكوين FIFO/LIFO**:
  1. انتقل إلى **المخزون** ← **طرق التقييم**
  2. عيّن المواد الخام إلى **FIFO**
  3. عيّن البضائع الجاهزة إلى **التكلفة المعيارية**
  4. كوّن الإنتاج تحت التشغيل إلى **التكلفة الفعلية**

  **عملية تجميع التكلفة**:
  1. شغّل حساب **تجميع التكلفة**
  2. حدّث التكاليف المعيارية ربع سنويًا
  3. راجع واعتمد تغييرات التكلفة

  **النتيجة المتوقعة**: منهجية تكلفة متسقة عبر جميع المنتجات.
  {{< /tab >}}

  {{< tab >}}
  **2. تحليل تباين الإنتاج**

  **جمع التكاليف الفعلية**:
  1. انتقل إلى **التصنيع** ← **تحليل التكلفة**
  2. اختر أوامر العمل المكتملة
  3. راجع التكاليف الفعلية مقابل المعيارية:
     - **استخدام المواد**: الفعلي مقابل كميات قائمة المواد
     - **كفاءة العمالة**: الساعات الفعلية مقابل المعيارية
     - **استيعاب النفقات العامة**: المطبّق مقابل الفعلي

  **حساب التباينات**:
  1. **تباينات المواد**:
     - تباين السعر: (السعر الفعلي - السعر المعياري) × الكمية الفعلية
     - تباين الاستخدام: (الكمية الفعلية - الكمية المعيارية) × السعر المعياري

  2. **تباينات العمالة**:
     - تباين المعدل: (المعدل الفعلي - المعدل المعياري) × الساعات الفعلية
     - تباين الكفاءة: (الساعات الفعلية - الساعات المعيارية) × المعدل المعياري

  3. **تباينات النفقات العامة**:
     - تباين الإنفاق: النفقات العامة الفعلية - النفقات العامة المدرجة في الميزانية
     - تباين الحجم: النفقات العامة المطبّقة - النفقات العامة المدرجة في الميزانية

  **النتيجة المتوقعة**: تحليل تباين مفصّل مع تحديد السبب الجذري.
  {{< /tab >}}

  {{< tab >}}
  **3. عملية تقييم المخزون**

  **الجرد المادي**:
  1. انتقل إلى **المخزون** ← **الجرد المادي**
  2. أنشئ أوراق الجرد حسب الموقع
  3. نفّذ الجرد المادي
  4. أدخل نتائج الجرد
  5. عالج تعديلات الجرد

  **تقييم المخزون**:
  1. **المواد الخام** (طريقة FIFO):
     - احسب متوسط التكلفة المرجّح
     - طبّق طبقات تكلفة FIFO
     - عدّل للتقادم

  2. **الإنتاج تحت التشغيل**:
     - تكاليف المواد المصروفة
     - تكاليف العمالة المطبّقة
     - تخصيص النفقات العامة

  3. **البضائع الجاهزة** (التكلفة المعيارية):
     - التكلفة المعيارية لكل وحدة
     - الكمية المتوفرة
     - احتياطيات التقادم

  **تقارير التقييم**:
  - تحليل تقادم المخزون
  - تقرير المخزون بطيء الحركة
  - تقييم التقادم
  - تحليل اتجاه التكلفة

  **النتيجة المتوقعة**: تقييم مخزون دقيق للتقارير المالية.
  {{< /tab >}}

  {{< tab >}}
  **4. تقارير التكلفة وتحليلها**

  **تحليل ربحية المنتج**:
  1. انتقل إلى **التقارير** ← **تحليل التكلفة**
  2. أنشئ **تقرير ربحية المنتج**:
     - إيرادات المبيعات حسب المنتج
     - التكلفة المعيارية للبضائع المباعة
     - تحليل الهامش الإجمالي
     - تحليل تأثير الحجم

  **تقارير تكلفة التصنيع**:
  1. **تقرير تكلفة الإنتاج**:
     - إجمالي التكاليف حسب الفترة
     - اتجاهات التكلفة لكل وحدة
     - مقاييس الكفاءة

  2. **تقرير ملخص التباين**:
     - تباينات المواد والعمالة والنفقات العامة
     - اتجاهات التباين عبر الزمن
     - تحليل الاستثناءات

  **لوحات معلومات الإدارة**:
  - مقاييس التكلفة في الوقت الفعلي
  - مؤشرات الأداء الرئيسية
  - مقارنة الميزانية مقابل الفعلي
  - الربحية حسب خط المنتج

  **توصيات تحسين التكلفة**:
  - تحديد فرص خفض التكلفة
  - اقتراحات تحسين العملية
  - مفاوضات تكلفة الموردين
  - ملاحظات التصميم القابل للتصنيع

  **النتيجة المتوقعة**: رؤى تكلفة شاملة لاتخاذ قرارات الإدارة.
  {{< /tab >}}
{{< /tabs >}}

### ميزات التكلفة المتقدمة

{{< callout type="info" >}}
**أدوات إدارة التكلفة**:
- **التكلفة على أساس الأنشطة**: تخصيص النفقات العامة بناءً على الأنشطة
- **التكلفة المستهدفة**: تحديد أهداف التكلفة للمنتجات الجديدة
- **تكلفة دورة الحياة**: تتبع التكاليف طوال دورة حياة المنتج
- **محاكاة التكلفة**: نمذجة تأثيرات التكلفة للتغييرات
{{< /callout >}}

## 🤝 سير العمل 6: عمليات التعاقد من الباطن {#subcontracting}

أدر عمليات التصنيع الخارجية بما في ذلك توريد المواد وشراء الخدمات.

### السيناريو: التعاقد من الباطن على عمليات تجميع SMT

**الهدف**: الاستعانة بمصدر خارجي لعمليات SMT (تقنية التركيب السطحي) لمورد خارجي مع الحفاظ على الجودة والتحكم في التكلفة.

**النتيجة المتوقعة**: عملية تعاقد من الباطن فعّالة مع رؤية وتحكم كاملين.

### تعليمات خطوة بخطوة

{{< tabs items="الإعداد,أمر الشراء,نقل المواد,الاستلام" >}}
  {{< tab >}}
  **1. إعداد التعاقد من الباطن**

  **تكوين المورد**:
  1. انتقل إلى **المشتريات** ← **الملف الرئيسي للمورد**
  2. أنشئ/حدّث المورد **SMT-Services-Ltd**
  3. عيّن نوع المورد إلى **متعاقد من الباطن**
  4. كوّن القدرات:
     - خدمات تجميع SMT
     - قدرات الاختبار
     - شهادات الجودة
     - معلومات الطاقة

  **إعداد صنف الخدمة**:
  1. انتقل إلى **المخزون** ← **الصنف الرئيسي**
  2. أنشئ صنف خدمة **SMT-ASSEMBLY-SERVICE**
  3. عيّن نوع الصنف إلى **خدمة**
  4. حدّد تخصيص مركز التكلفة
  5. عيّن معدل الخدمة المعياري

  **تكوين قائمة المواد**:
  1. حدّث قائمة المواد لـ **AC-SENSOR-001**
  2. عدّل **العملية 10** (تجميع SMT):
     - عيّن نوع العملية إلى **متعاقد من الباطن**
     - عيّن مورد التعاقد من الباطن
     - حدّد استهلاك المواد لدى المورد

  **النتيجة المتوقعة**: تم تكوين إطار عمل كامل للتعاقد من الباطن.
  {{< /tab >}}

  {{< tab >}}
  **2. إنشاء أمر شراء للتعاقد من الباطن**

  **إنشاء أمر شراء للتعاقد من الباطن**:
  1. انتقل إلى **التصنيع** ← **أوامر التعاقد من الباطن**
  2. انقر على **إنشاء من أمر العمل**
  3. اختر أمر العمل لـ **AC-SENSOR-001**
  4. ينشئ النظام أمر شراء مع:
     - سطر الخدمة: خدمة تجميع SMT
     - أسطر المواد: المكونات المراد توريدها
     - جدول التسليم
     - متطلبات الجودة

  **تكوين تفاصيل أمر الشراء**:
  1. **سطر الخدمة**:
     - الكمية: 100 وحدة
     - المعدل: 2.50 دولار لكل وحدة
     - تاريخ التسليم: الأربعاء القادم

  2. **أسطر المواد** (المراد توريدها):
     - لوحات PCB: 100 قطعة
     - مكونات SMT: حسب متطلبات قائمة المواد
     - موقع التسليم: منشأة المورد

  3. **الشروط والأحكام**:
     - مواصفات الجودة
     - متطلبات الفحص
     - شروط الدفع
     - إجراءات إعادة المواد

  **النتيجة المتوقعة**: أمر شراء شامل للتعاقد من الباطن جاهز للموافقة.
  {{< /tab >}}

  {{< tab >}}
  **3. نقل المواد إلى المتعاقد من الباطن**

  **تحضير شحنة المواد**:
  1. انتقل إلى **المخزون** ← **نقل المواد**
  2. أنشئ أمر نقل إلى موقع المورد
  3. ضمّن جميع المواد حسب أمر الشراء:
     - تحقق من مطابقة الكميات للمتطلبات
     - تحقق من حالة جودة المواد
     - أنشئ قوائم التعبئة

  **تنفيذ صرف المواد**:
  1. **انتقاء المواد**:
     - أنشئ قائمة الانتقاء
     - احجز المخزون
     - عالج الانتقاء المادي

  2. **فحص الجودة**:
     - افحص المواد قبل الشحن
     - تحقق من أرقام الأجزاء والكميات
     - تحقق من عدم وجود ضرر

  3. **الشحن إلى المورد**:
     - أنشئ مستندات الشحن
     - تتبّع حالة الشحنة
     - أكّد استلام المورد

  **تأثير المخزون**:
  - نُقلت المواد إلى موقع "بضاعة أمانة لدى المورد"
  - احتفظ بالملكية حتى الاستهلاك
  - تتبّع المخزون في منشأة المورد

  **النتيجة المتوقعة**: تُسلَّم المواد إلى المتعاقد من الباطن مع إمكانية تتبع كاملة.
  {{< /tab >}}

  {{< tab >}}
  **4. الاستلام ومراقبة الجودة**

  **استلام البضائع المتعاقد عليها من الباطن**:
  1. انتقل إلى **المشتريات** ← **استلام البضائع**
  2. اختر أمر شراء التعاقد من الباطن
  3. سجّل تفاصيل الاستلام:
     - الكمية المستلمة
     - تاريخ ووقت التسليم
     - مرجع مذكرة تسليم المورد
     - الفحص البصري الأولي

  **فحص الجودة**:
  1. **الفحص الوارد**:
     - أنشئ أمر فحص
     - نفّذ اختبارات الجودة حسب الخطة
     - سجّل نتائج الفحص
     - اتخذ قرار القبول/الرفض

  2. **التحقق من الخدمة**:
     - تحقق من العمل المنجز حسب المواصفات
     - تحقق من جودة التجميع
     - تحقق من نتائج الاختبار من المورد
     - وثّق أي تباينات

  **إكمال العملية**:
  1. **قبول البضائع**:
     - انقل إلى موقع المخزون المناسب
     - حدّث تقدم أمر العمل
     - أطلق العملية التالية

  2. **معالجة الفاتورة**:
     - طابق الفاتورة مع أمر الشراء والاستلام
     - تحقق من رسوم الخدمة
     - عالج الدفع

  3. **تخصيص التكلفة**:
     - خصّص تكاليف الخدمة لأمر العمل
     - حدّث التكاليف المعيارية للمنتج
     - سجّل تباين الفعلي مقابل الميزانية

  **النتيجة المتوقعة**: تُستلم الأجزاء المتعاقد عليها من الباطن بجودة عالية وتكون جاهزة للعملية التالية.

## 📈 مؤشرات أداء التصنيع والتحليلات

راقب مؤشرات الأداء الرئيسية لتحسين عمليات التصنيع.

### مقاييس التصنيع الأساسية

{{< tabs items="مؤشرات أداء الإنتاج,مقاييس الجودة,تحليل التكلفة,الكفاءة" >}}
  {{< tab >}}
  **أداء الإنتاج**

  **المقاييس الرئيسية للتتبع**:
  - **الفعالية الإجمالية للمعدات (OEE)**
    - التوفر × الأداء × الجودة
    - الهدف: >85% للمعدات الحرجة

  - **التسليم في الوقت المحدد (OTD)**
    - الأوامر المسلّمة في التاريخ الموعود
    - الهدف: >95% لرضا العملاء

  - **إنتاجية الإنتاج**
    - الوحدات المنتجة في الساعة/اليوم
    - تتبّع الاتجاهات والتحسينات

  - **استغلال الطاقة**
    - الإنتاج الفعلي مقابل الطاقة المخططة
    - تحديد نقاط الاختناق والقيود

  **الوصول إلى لوحة المعلومات**: التصنيع ← التحليلات ← لوحة معلومات الإنتاج
  {{< /tab >}}

  {{< tab >}}
  **أداء الجودة**

  **مقاييس الجودة**:
  - **العائد من المرور الأول (FPY)**
    - الوحدات الناجحة من المرة الأولى / إجمالي الوحدات
    - الهدف: >98% للتميز في التصنيع

  - **معدل العيوب**
    - العيوب لكل مليون فرصة
    - التتبع حسب المنتج والعملية

  - **مرتجعات العملاء**
    - معدل المرتجعات والأسباب الجذرية
    - تحليل تكلفة الجودة

  - **جودة المورد**
    - نتائج الفحص الوارد
    - بطاقات أداء المورد

  **الوصول**: الجودة ← التحليلات ← لوحة معلومات الجودة
  {{< /tab >}}

  {{< tab >}}
  **تحليلات التكلفة**

  **أداء التكلفة**:
  - **تحليل تباين التكلفة**
    - تباينات المواد والعمالة والنفقات العامة
    - الاتجاه والتحليل الشهري

  - **ربحية المنتج**
    - الهامش الإجمالي حسب المنتج
    - تحسين مزيج المنتجات

  - **دوران المخزون**
    - نسبة دوران المخزون
    - كفاءة رأس المال العامل

  - **اتجاهات التكلفة لكل وحدة**
    - تغييرات تكلفة الوحدة عبر الزمن
    - تتبع مبادرات خفض التكلفة

  **الوصول**: المحاسبة ← التحليلات ← لوحة معلومات التكلفة
  {{< /tab >}}

  {{< tab >}}
  **الكفاءة التشغيلية**

  **مقاييس الكفاءة**:
  - **كفاءة العمالة**
    - الساعات المعيارية مقابل الساعات الفعلية
    - تحسينات الإنتاجية

  - **استغلال الآلة**
    - وقت تشغيل المعدات واستخدامها
    - فعالية الصيانة

  - **استغلال المواد**
    - نسب عائد المواد
    - تتبع تقليل الهدر

  - **أداء وقت التوريد**
    - أوقات توريد التصنيع
    - سرعة تلبية الأوامر

  **الوصول**: التصنيع ← التحليلات ← لوحة معلومات الكفاءة
  {{< /tab >}}
