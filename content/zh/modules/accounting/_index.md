会计模块提供全面的会计解决方案，配备丰富的专业交易处理应用，覆盖财务运营的各个环节。本模块超越基础财务会计，为复杂业务场景提供专业化的交易处理能力。

## 概述

会计模块提供：
- **完整的交易处理** - 丰富的专业交易应用集合
- **高级财务运营** - 从基础会计延伸至复杂的财务工作流
- **专业行业支持** - 针对特定行业定制的交易应用
- **一体化工作流管理** - 顺畅的交易处理流程
- **全面的审计轨迹** - 完整的交易追踪与合规

{{< callout type="info" >}}
**模块范围**：本模块涵盖 BigLedger 平台上的全部交易处理应用，是目前最全面的会计解决方案。
{{< /callout >}}

## 集成架构

### 交易流
```
源交易 → 处理应用 → 总账 → 财务报告
```

### 数据依赖
```
核心模块（主数据） → 交易处理应用 → 总账 → 财务报表与分析
```

### 实时集成
- 所有交易应用实时更新总账
- 跨模块校验确保数据完整性
