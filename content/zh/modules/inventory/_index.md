库存模块提供全面的库存管理解决方案，覆盖库存控制的方方面面，从基础的物品追踪到高级库存优化。本模块支持单一或多地点、从简单到复杂库存需求的各类企业。

## 概述

库存模块提供：
- **完整的库存管理** - 实时库存追踪与控制
- **多地点支持** - 跨多个仓库的集中管控
- **高级计价方法** - FIFO、LIFO、加权平均、标准成本
- **库存移动追踪** - 所有库存交易的完整审计轨迹
- **库存优化** - 自动补货与库存水平优化
- **随时可集成** - 与销售、采购、会计模块无缝集成

{{< callout type="info" >}}
**模块定位**：本模块提供精细化的库存管理能力，适合有库存需求的制造商、经销商、零售商与服务型企业。
{{< /callout >}}

## 核心功能

### 库存管理基础
- **实时库存追踪** - 全部地点的实时库存水平
- **多单位管理** - 支持不同计量单位与换算
- **序列号/批次追踪** - 序列化与批次物品的完整可追溯性
- **库存计价** - 多种成本核算方法与自动计价
- **库存分类** - 层级化的产品分类与分组

### 库存操作
- **库存调整** - 实物盘点调整与差异管理
- **库存调拨** - 跨地点与跨公司调拨
- **库存预留** - 为销售订单与生产分配库存
- **组装/拆装** - 物料清单与套件管理
- **循环盘点** - 永续盘点与对账

### 库存分析
- **库存水平分析** - 最小/最大水平、补货点、安全库存
- **移动分析** - 快销/慢销商品与周转率
- **计价报表** - 按地点、类别与方法统计库存价值
- **库龄分析** - 库存老化与呆滞追踪
- **盈利分析** - 按产品与类别的毛利分析

### 采购集成
- **自动采购** - 补货点触发与采购建议
- **供应商管理** - 供应商绩效与交期追踪
- **采购订单集成** - 从 PO 到收货的无缝处理
- **成本管理** - 到岸成本计算与分摊
- **质量控制** - 来料检验与质量工作流

## 实施路径

### 第一阶段：基础搭建（第 1-2 周）
1. **库存结构**
   - 定义产品类别与层级
   - 设置库存地点与仓库
   - 配置计量单位与换算
   - 定义库存计价方法

2. **商品主数据准备**
   - 创建库存商品主数据
   - 设置产品编码与描述
   - 配置库存参数与策略
   - 定义补货水平与安全库存

### 第二阶段：库存操作（第 3-4 周）
3. **库存移动配置**
   - 建立库存调整流程
   - 配置调拨工作流
   - 落地循环盘点程序
   - 建立质量控制流程

4. **集成搭建**
   - 配置采购集成
   - 建立销售订单集成
   - 落地会计集成
   - 配置自动化工作流

### 第三阶段：分析与优化（第 5-6 周）
5. **报表与分析**
   - 配置库存报表
   - 搭建仪表盘分析
   - 落地 ABC 分析
   - 配置绩效指标

6. **上线准备**
   - 数据迁移与验证
   - 用户培训与测试
   - 性能优化
   - 流程文档编制

## 业务流程

### 库存收货流程
```
采购订单 → 收货 → 质量检验 → 库存更新 → 供应商付款
```

### 库存发货流程
```
销售订单 → 库存预留 → 拣货单 → 发货 → 库存更新
```

### 循环盘点流程
```
盘点计划 → 盘点执行 → 差异分析 → 调整 → 对账
```

## 库存计价方法

### 成本核算方法
- **先进先出（FIFO）** - 按收货顺序出库
- **后进先出（LIFO）** - 最近收货的物品优先出库
- **加权平均** - 在库全部物品的平均成本
- **标准成本** - 预设成本并进行差异分析
- **移动平均** - 持续更新的平均成本

### 计价功能
- **多成本层** - 按收货批次的精细成本追踪
- **成本调整** - 手动与自动成本修正
- **到岸成本分摊** - 运费、关税与间接费用分摊
- **重估处理** - 期末库存重估
- **多币种处理** - 多币种成本追踪

## 故障排查指南

### 常见问题

**库存余额不准确**
- 核实所有交易已过账
- 检查待处理的调整单
- 核对跨地点调拨
- 验证期初余额

**成本核算问题**
- 检查计价方法配置
- 核查成本层完整性
- 核实到岸成本分摊
- 确认期末处理已完成

**性能问题**
- 优化数据库索引
- 检查交易量
- 排查并发用户负载
- 考虑数据归档
