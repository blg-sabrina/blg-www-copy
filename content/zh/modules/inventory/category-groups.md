类别组为在 BigLedger 中组织库存类别提供了层级结构。该系统有助于在整个产品目录中保持一致性，并简化库存管理。

## 概述

在类别组下，用户可以创建和编辑类别，并指定类别组与各个类别之间的关系。在项目菜单中创建项目时，需指定其类别分配。

{{< callout type="info" >}}
**注意**：类别组和项目类别既可以通过单据项目维护应用程序管理，也可以通过库存项目维护应用程序管理。每个类别都关联到一个类别组，以便更好地组织。
{{< /callout >}}

## 创建类别组

### 步骤 1：访问类别组

1. 导航至 **库存** > **类别组**
2. 点击 **"+"** 按钮创建新的类别组
3. 将打开第三个容器用于创建类别组

### 步骤 2：填写类别组详细信息

在主选项卡中完成以下必填字段：

| 字段 | 描述 | 示例 |
|-------|-------------|---------|
| **类别组代码** | 组的唯一标识符 | `ELEC`、`FURN`、`FOOD` |
| **类别组名称** | 组的描述性名称 | `Electronics`、`Furniture`、`Food & Beverage` |
| **类型** | 分类类型 | `Product`、`Service`、`Raw Material` |
| **参数代码** | 用于系统集成的参数代码 | 可选 |
| **参数名称** | 参数描述 | 可选 |
| **状态** | 启用/停用状态 | `Active` |

### 步骤 3：保存类别组

点击 **保存** 以创建类别组。系统将验证是否存在重复代码及必填字段。

## 编辑类别组

### 进入编辑模式

1. 点击列表中的任意类别组
2. 打开第二个容器，显示类别组详细信息
3. 提供两个选项卡供编辑

### 主选项卡

主选项卡显示在类别创建期间输入的所有信息：

- 编辑类别组属性
- 更新状态（启用/停用）
- 删除类别组（如果没有关联的类别）

{{< callout type="warning" >}}
**警告**：删除类别组将影响所有关联的类别。删除前请确保没有活跃项目正在使用这些类别。
{{< /callout >}}

### 类别选项卡

类别选项卡显示组内所有关联的类别：

- 查看组中的所有类别
- 添加新类别
- 编辑现有类别
- 移除类别关联

## 管理类别

### 创建类别

要将类别添加到组：

1. 打开类别组
2. 导航至 **类别** 选项卡
3. 点击 **"+"** 按钮
4. 填写类别详细信息：
   - 类别代码
   - 类别名称
   - 描述
   - 排序顺序
   - 状态

### 编辑/删除类别

通过点击列表中的任意类别来访问类别详细信息：

1. **主选项卡**：编辑类别属性
   - 更新类别信息
   - 更改状态
   - 删除类别

2. **管理图像选项卡**：处理类别图像
   - 上传类别图像
   - 替换现有图像
   - 支持多种图像格式（JPG、PNG、GIF）
   - 推荐图像尺寸：500x500 像素

## 最佳实践

### 命名约定

- 使用一致的编码模式（例如 `ELEC-001`、`ELEC-002`）
- 保持名称描述清晰但简洁
- 避免在代码中使用特殊字符

### 类别层级

示例结构：
```
Electronics (ELEC)
├── Computers (ELEC-COMP)
│   ├── Laptops (ELEC-COMP-LAP)
│   ├── Desktops (ELEC-COMP-DSK)
│   └── Tablets (ELEC-COMP-TAB)
├── Mobile Devices (ELEC-MOB)
│   ├── Smartphones (ELEC-MOB-PHN)
│   └── Accessories (ELEC-MOB-ACC)
└── Audio/Video (ELEC-AV)
    ├── Televisions (ELEC-AV-TV)
    └── Sound Systems (ELEC-AV-SND)
```

### 维护提示

1. **定期审查**：定期审查类别结构
2. **合并**：合并相似类别以避免重复
3. **文档**：维护类别指南供员工参考
4. **培训**：确保员工理解类别分配规则

## 与其他模块的集成

类别组与以下模块集成：

- **库存管理**：产品分类
- **采购**：供应商类别偏好
- **销售**：基于类别的促销
- **报告**：类别绩效分析
- **电子商务**：在线商店导航

## 故障排除

### 无法删除类别组
- 检查是否有关联的类别
- 确认没有活跃项目使用这些类别
- 审查任何待处理的交易

### 类别未出现在项目创建中
- 确认类别状态为启用
- 检查类别组状态
- 验证用户权限

### 图像上传问题
- 检查文件格式（JPG、PNG、GIF）
- 验证文件大小（推荐 < 5MB）
- 清除浏览器缓存
