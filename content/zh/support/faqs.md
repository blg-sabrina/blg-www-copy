## 电子商务与 EcomSync

### EcomSync 系统支持多少个市场平台？

目前，EcomSync 支持：
- **Lazada** - 与库存和订单管理的完整集成
- **Shopee** - 完整的市场同步
- **Shopify** - 电子商务网站集成
- **WooCommerce** - WordPress 电子商务平台
- 更多市场平台即将推出！

### 我如何更新像 11.11 或 12.12 这样的促销活动？

在销售订单 Applet 中：
1. 导航到商品列表
2. 转到市场平台标签页
3. 为特定时段设置特别促销价格
4. 系统将自动将这些价格同步到所有已连接的市场平台

### 我如何在各市场平台之间保持库存可用性？

库存可用性由中央集中控制：
1. 转到市场平台库存可用性标签页
2. 为每个市场平台设置可用数量
3. 配置缓冲库存水平
4. 启用自动库存同步

### 订单状态在 Applet 中如何更新？

订单状态实时同步：
- 系统自动跟随市场平台状态
- 更新即时反映在所有平台上
- 状态变更在 BigLedger 中触发相应的工作流

### 如果卖家取消订单会怎样？

订单取消得到无缝处理：
- 通过任何市场平台（Lazada、Shopee 等）的取消会在各处反映
- 通过 EcomSync 平台的取消会更新所有已连接的渠道
- 库存自动调整
- 自动发送客户通知

### 我可以控制通过 EcomSync 接收多少订单吗？

可以，您可以配置订单处理限制：
- 设置每个时段的最大订单数
- 配置处理队列
- 使用调度程序管理订单接收
- 为高流量时段实施速率限制

### 平台能处理高订单流量时段吗？

当然！该系统专为峰值性能而设计：
- 订单在优化的队列中处理
- 可配置的订单处理速率
- 高峰时段自动扩展
- 内置调度程序用于负载分配

### 产品属性可以定制吗？

可以，属性完全可定制：
- 根据用户要求创建自定义属性
- 映射市场平台特定属性
- 批量更新功能
- 基于模板的属性管理

## BigLedger 一般问题

### BigLedger 支持哪些行业？

BigLedger 支持各种行业，包括：
- 零售与餐饮
- 批发与分销
- 制造业
- 汽车与维修厂
- 专业服务
- 医疗保健

### BigLedger 符合马来西亚法规吗？

是的，BigLedger 完全符合：
- GST/SST 要求
- LHDN MyInvois 电子发票
- MDEC PEPPOL 标准
- 马来西亚会计准则

### 系统要求是什么？

BigLedger 基于云端，需要：
- 现代网页浏览器（Chrome、Firefox、Safari、Edge）
- 稳定的互联网连接
- 无特殊硬件要求
- 提供 iOS 和 Android 移动应用

### 我的数据有多安全？

BigLedger 实施企业级安全：
- 256 位 SSL 加密
- 每日自动备份
- 符合 ISO 27001 的数据中心
- 基于角色的访问控制
- 提供双重身份验证

### 我可以将 BigLedger 与其他系统集成吗？

可以，BigLedger 提供：
- 用于自定义集成的 REST API
- 面向热门服务的预构建连接器
- 用于实时事件的 Webhook 支持
- 数据导入/导出功能

### 提供哪些培训？

我们提供全面的培训：
- 在线视频教程
- 实时网络研讨会
- 面向企业的现场培训
- 认证计划
- 用户文档和指南

### 定价如何构成？

BigLedger 提供灵活的定价：
- 按月或按年订阅
- 基于用户的定价层级
- 基于模块的定价选项
- 提供批量折扣
- 免费试用期

### 提供哪些支持选项？

多个支持渠道：
- 电子邮件支持
- 电话支持
- WhatsApp 支持
- 在线聊天
