掌握 BigLedger 中的手动和自动日记账分录。

## 概述

日记账分录是 BigLedger 中会计核算的基础，它将所有财务交易记录到总账中。

## 日记账分录类型

### 手动日记账
- 调整分录
- 应计和递延
- 更正
- 期初余额

### 自动日记账
- 销售发票
- 采购账单
- 薪资过账
- 折旧

### 经常性日记账
- 每月应计
- 租金费用
- 保险
- 订阅

## 创建日记账分录

### 步骤 1：访问日记账模块

导航至 **Finance** → **Ledger and Journal**

### 步骤 2：创建新日记账

1. 点击 **New Journal Entry**
2. 选择日记账类型：
   - 普通日记账
   - 付款日记账
   - 收款日记账
   - 采购日记账
   - 销售日记账

### 步骤 3：输入表头信息

- **日记账编号**：自动生成或手动录入
- **日期**：交易日期
- **参考**：外部单据参考
- **描述**：日记账描述
- **币种**：交易币种

### 步骤 4：添加日记账行

对于每一行：
```
Account | Description | Debit | Credit | Tax | Project
--------|-------------|-------|--------|-----|--------
1111    | Cash receipt| 1000  |        |     | 
4000    | Sales revenue|      | 1000   | SST | Project A
```

### 步骤 5：验证并过账

1. 核实借方 = 贷方
2. 检查税额计算
3. 审核科目代码
4. 过账日记账分录

## 常见日记账分录

### 销售交易
```
Dr. Accounts Receivable  1,060
  Cr. Sales Revenue         1,000
  Cr. SST Payable             60
```

### 采购交易
```
Dr. Expense Account       500
Dr. Input Tax              30
  Cr. Accounts Payable      530
```

### 银行收款
```
Dr. Bank Account        1,000
  Cr. Accounts Receivable  1,000
```

### 折旧
```
Dr. Depreciation Expense  100
  Cr. Accumulated Depreciation 100
```
