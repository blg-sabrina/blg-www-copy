借助 BigLedger 的报表套件生成准确的财务报告。

## 可用报表

### 核心财务报表

#### 资产负债表
显示特定日期的财务状况
- 资产
- 负债
- 权益

#### 损益表
显示某一期间的财务业绩
- 收入
- 费用
- 净利润/亏损

#### 现金流量表
显示现金变动
- 经营活动
- 投资活动
- 筹资活动

#### 权益变动表
显示权益变动
- 股本
- 留存收益
- 其他储备

## 生成报表

### 快速生成报表

1. 导航至 **Finance** → **Financial Report Applet**
2. 选择报表类型
3. 选择参数：
   - 期间（月/季/年）
   - 公司/分支机构
   - 币种
   - 比较期间
4. 点击 **Generate**

### 报表参数

**日期范围**
- 当前期间
- 年初至今
- 自定义范围
- 比较期间

**格式选项**
- 汇总与明细
- 合并与单体
- 本位币与报告币
- 图形与表格

## 自定义报表

### 报表生成器

创建自定义报表：
1. 定义报表结构
2. 将科目映射到行
3. 添加计算
4. 格式化呈现方式
5. 保存模板

### 自定义公式

示例：
```
Gross Profit = Revenue - COGS
Gross Margin % = (Gross Profit / Revenue) * 100
Current Ratio = Current Assets / Current Liabilities
```

## 最佳实践

### 月末流程
1. 完成所有交易
2. 执行对账
3. 过账调整分录
4. 生成报表草稿
5. 审核并批准
6. 定稿并分发

### 报表审核
- 检查完整性
- 核实准确性
- 审查差异
- 验证计算
- 确认合规性
