## 当前基本功能

### 创建成员类别

1. 从主模块列表中导航到 **Member Class**
2. 单击 **"+"** 按钮创建新的成员类别
3. 填写必填信息：
   - **Membership Class Code**：系统生成的标识符
   - **Membership Class Name**：角色的描述性名称

### 管理现有类别

- **编辑**：修改成员类别名称和状态
- **删除**：移除未使用的成员类别
- **列表视图**：浏览所有已配置的成员分类
