了解如何将产品和服务添加到 BigLedger 目录中，并进行正确的配置，以实现最佳的库存管理和电子商务集成。

## 概述

项目是 BigLedger 中业务运营的基础。无论您销售的是实物产品、数字服务还是捆绑产品，正确的项目设置都能确保准确的库存跟踪、定价以及跨所有销售渠道的集成。

{{< callout type="info" >}}
**开始之前**：确保您拥有创建项目所需的权限。如果您看不到"+"按钮或创建选项，请联系您的系统管理员。
{{< /callout >}}

---

## 分步创建流程

### 1. 访问项目创建界面

1. 从主菜单导航到 **Item Management**
2. 单击 **"+" 按钮** 或 **"Create New Item"** 按钮
3. 创建项目对话框将在新容器中打开

### 2. 填写必填信息

填写以下必填字段以创建您的项目：

#### 基本详情

**项目代码 (Item Code)**
- 由系统自动生成
- 用于内部跟踪和识别
- 创建后无法修改

**项目名称 (Item Name)** *（必填）*
- 为您的产品或服务提供清晰、描述性的名称
- 这将出现在所有交易和报告中
- 最佳实践：使用一致的命名约定

**类型 (Type)** *（必填）*
- 根据项目的性质从下拉菜单中选择：
  - **Product**：您销售或制造的实物商品
  - **Service**：您提供的无形服务
  - **Bundle**：产品和/或服务的组合
  - **Digital**：可下载或数字产品

**基本计量单位 (Base Unit of Measure, UOM)** *（必填）*
- 选择跟踪此项目的最小单位
- 常见示例：单位、千克、米、升、小时、件
- 项目创建后无法更改
- 所有交易都将基于此 UOM

#### 可选信息

**项目描述 (Item Description)**
- 供内部参考的详细描述
- 有助于复杂产品或特殊处理说明
- 出现在报告中，并可协助处理客户咨询

### 3. 配置电子商务集成

**EcomSync 集成**
- 勾选 **"EcomSync"** 复选框以启用多渠道同步
- 这会将项目同步到 EcomSync Applet 以进行在线商店集成
- 建议为您计划在线销售的所有项目启用

{{< callout type="tip" >}}
**电子商务提示**：始终为零售项目启用 EcomSync，即使您目前不在线销售。这可以为您未来扩展到电子商务平台时的目录做好准备。
{{< /callout >}}

### 4. 保存并验证

1. 单击 **"Save"** 或 **"Create Item"** 以保存您的新项目
2. 成功消息将确认项目已创建
3. 您将被自动重定向到 **Edit Item** 页面
4. 在继续之前验证所有信息是否正确

## 创建后：后续步骤

项目成功创建后，您可以：

### 即时操作
- **编辑其他详情**：添加定价、类别和规格
- **上传图片**：为电子商务集成添加产品照片
- **设置定价**：配置定价方案和客户专属价格
- **配置类别**：组织项目以便更好地管理

### 高级配置
- **多地点设置**：为不同仓库配置库存水平
- **供应商信息**：链接供应商和采购成本
- **条形码生成**：为 POS 和仓库操作创建条形码
- **税务配置**：设置适当的税率和代码

---

## 最佳实践

### 命名约定
- 使用一致的、描述性的名称
- 在相关时在名称中包含关键规格
- 避免使用可能导致集成问题的特殊字符

### 组织策略
- 在添加大量项目之前创建合理的类别结构
- 在类似项目中使用一致的 UOM 选择
- 如果使用自定义编号，请规划您的项目代码

### 电子商务优化
- 始终为可销售项目启用 EcomSync
- 准备高质量的产品图片
- 撰写客户友好的描述

## 常见问题及解决方案

**问题**：看不到创建按钮
**解决方案**：与您的管理员确认您的用户权限

**问题**：EcomSync 复选框呈灰色
**解决方案**：确保 EcomSync applet 已安装并激活

**问题**：UOM 选项有限
**解决方案**：联系您的管理员以添加自定义计量单位
